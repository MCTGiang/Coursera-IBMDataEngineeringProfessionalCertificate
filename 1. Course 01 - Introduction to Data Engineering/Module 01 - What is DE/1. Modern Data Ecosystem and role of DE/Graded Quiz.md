### 1. A modern data ecosystem includes a network of continually evolving entities. It includes: 
    A. Data sources, enterprise data repository, business stakeholders, and tools, applications, and infrastructure to manage data
    B. Data sources, databases, and programming languages
    C. Data providers, databases, and programming languages
    D. Social media sources, data repositories, and APIs

_The answer is A. Data sources, enterprise data repository, business stakeholders, and tools, applications, and infrastructure to manage data_
### 2. Data Engineers work within the data ecosystem to: 
    A. Analyze data for deriving insights
    B. Provide business intelligence solutions by monitoring data on different business functions
    C. Analyze data for actionable insights
    D. Develop and maintain data architectures

_The answer is D. Develop and maintain data architectures_
### 3. The goal of data engineering is to make quality data available for fact-finding and decision-making. Which one of these statements captures the process of data engineering? 
    A. Collecting, processing, and storing data
    B. Collecting, processing, storing, and making data available to users securely
    C. Processing data and making it available to users securely
    D. Collecting, processing, and making data available to users securely

_The answer is B. Collecting, processing, storing, and making data available to users securely_
### 4. Data extracted from disparate sources can be stored in: 
    A. Data Lakes only
    B. Data Warehouses only
    C. Databases only
    D. Databases, data warehouses, data lakes, or any other type of data repository

_The answer is D. Databases, data warehouses, data lakes, or any other type of data repository_
